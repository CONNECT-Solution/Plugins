•	CONNECT-AdapterDocumentAssemblyProxy
•	CONNECT-AdapterDocumentRepository
•	CONNECT-AdapterMpi
•	CONNECT-DocumentManager
•	CONNECT-Hub
•	CONNECT-NHINAdapterService
•	CONNECTUniversalClientGUI
•	FHIRAdapters
•	FHIRRestServer
•	FHIRSoapTests
•	GenericFileTransfer
•	HPD-WS-Client
•	PerformanceTestTools
•	XDSBPlugin
